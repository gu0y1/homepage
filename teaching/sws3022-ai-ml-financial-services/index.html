This workshop introduces students to financial services, trading, and the importance of AI/ML in the fintech industry through a set of case studies.

Students learn fundamental AI/ML concepts, including supervised and unsupervised learning, Naive Bayes, logistic regression, bias-variance tradeoff, principal component analysis, support vector machines, decision trees, random forests, K-Means, agglomerative clustering, ARIMA, GARCH, evaluation metrics, and neural networks including ANN, CNN, RNN, LSTM, NLP, and LLMs.

The workshop provides hands-on experience in obtaining financial data via Quandl or Yahoo Finance, understanding financial data, and structuring data so that it is amenable to machine learning algorithms.

Students implement machine learning algorithms to extract key features from financial datasets. They are also trained to develop frontend dashboards using tools such as Streamlit and Dash, and to develop fintech web applications using ReactJS, Python Flask, and basic database operations.

The workshop also introduces future financial systems such as Finternet through case studies, research, and group discussions on future opportunities and applications.

Learning Outcomes

  • Understand and appreciate the growing importance of AI/ML in the financial industry.
  • Understand future financial systems such as Finternet and brainstorm the role of AI/ML in those systems.
  • Distinguish between supervised ML, unsupervised ML, deep learning, and artificial intelligence.
  • Understand statistical and mathematical models and their limitations.
  • Understand financial datasets and prepare data for ML using Python libraries.
  • Build and apply appropriate AI/ML models and data processing techniques for business decisions in financial settings.
  • Use ReactJS, Python Flask, and basic database operations to build fintech web applications.

Prerequisites

Basic Python programming knowledge, including basic data structures and NumPy.
